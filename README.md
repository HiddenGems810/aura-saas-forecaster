## What is AURA?

AURA is a **premium, single-page SaaS forecasting tool** designed for founders who demand boardroom-level financial clarity without enterprise-level overhead.

Drag the sliders. Watch your wealth trajectory render in real time. Export a full **6-page financial whitepaper PDF** in one click.

No sign-up. No paywalls. No compromise.

## The $MRR$ Logic Engine

AURA's projection core is a recursive SaaS compounding formula:

```
MRR_n = MRR_(n-1) × (1 + g/100 − c/100)
```

Where:
- `g` = Monthly Growth Rate (%)
- `c` = Monthly Churn Rate (%)
- `n` = Month index

Each month's output becomes the next month's input — creating the exponential curve. The **Cumulative Total** in the hero metric is the running sum of all monthly revenues, not just an annualized end-state.

The engine models:
- **Net growth factor** (growth minus churn) — the honest picture
- **7-Figure Milestone** detection — month when cumulative crosses $1M
- **Industry Benchmark overlay** — 3% growth / 5% churn median comparison
- **URL-synced state** — every scenario is a shareable link


Open [http://localhost:3000](http://localhost:3000) in your browser.
